Reddit Crypto Sentiment Analysis

This project involved scraping data from cryptocurrency-related subreddits to perform sentiment analysis. The goal was to classify posts and comments as positive, negative, or neutral to gauge market sentiment.

Key Activities:

  • Data collection from Reddit using PRAW.
  • Extensive text preprocessing (tokenization, stopword removal, lemmatization).
  • Sentiment analysis using TF-IDF vectorization and ML models like Logistic Regression and Naive Bayes.
  • Model performance evaluation using accuracy, confusion matrices, and classification reports.
  • Created an interactive Tableau dashboard for visualization.

Technologies Used: Python, NLTK, Scikit-learn, Matplotlib, Tableau, PRAW

Automated Short Answer Grading System

Developed a machine learning system for the automated grading of short text answers, leveraging NLP and ensemble learning techniques to achieve high accuracy.

Key Achievements:

  • Engineered over 50 features using Sentence-Transformers, Word2Vec, TF-IDF, ROUGE, and LDA.
  • Trained 11 different regressor models including SVR, XGBoost, and LightGBM.
  • Achieved a high correlation (0.85+ Pearson) through a weighted ensemble model.
  • Developed a complete pipeline with feature selection, scaling, and model serialization.

Technologies Used: Python, Scikit-learn, XGBoost, LightGBM, NLTK, spaCy
